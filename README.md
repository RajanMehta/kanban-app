# Kanban Task Board

A full-stack Kanban board for managing tasks across **To Do → In Progress → Done**.

- **Backend:** .NET 10 Web API, EF Core, SQLite
- **Frontend:** React + TypeScript (Vite)
- **Containerization:** Docker + docker-compose

> Setup and run instructions are added as each part is built.

## Project structure

```
kanban-app/
├── backend/            # .NET 10 Web API solution
│   ├── Kanban.slnx
│   └── Kanban.Api/
├── frontend/           # React + TypeScript app
└── docker-compose.yml
```
